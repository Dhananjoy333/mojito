# 🍸 Velvet Pour

**Velvet Pour** is a modern, animated bar website focused on showcasing premium drinks with an immersive visual experience. Built with **Next.js** and powered by **GSAP animations**, the site delivers smooth transitions, scroll-based storytelling, and an elegant UI inspired by luxury cocktail bars.

---

## ✨ Features

- 🎬 **Scroll-Based Background Video**
  - A high-quality **mojito drink video** plays dynamically in the background
  - Video syncs with scroll for a cinematic feel

- 🎯 **Animated Hero Section**
  - Eye-catching hero section with GSAP-powered animations
  - Smooth text reveals and transitions

- 🍹 **Drinks Showcase**
  - 4 curated drinks with:
    - Recipe details
    - Ingredients
    - Visual presentation
  - Swipe/slide interaction to explore drinks

- ⚡ **GSAP Animations**
  - ScrollTrigger-based animations
  - Seamless transitions between sections
  - Performance-optimized animations

- 🧭 **Structured Sections**
  - Home
  - Hero (Animated)
  - Cocktails (Available drinks and prices)
  - About
  - Art
  - Drinks (Interactive)
  - Contact

- 📱 **Responsive Design**
  - Works across desktop and mobile devices

---

## 🛠️ Tech Stack

- **Framework:** Next.js
- **Animation:** GSAP (GreenSock Animation Platform)
- **Styling:** Tailwind CSS 
- **Media:** HTML5 Video
